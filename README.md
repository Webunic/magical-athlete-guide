# Magical Athlete Companion

Мобильный русскоязычный помощник для настольной игры **Magical Athlete**.

**Приложение:** https://magical-athlete-guide.netlify.app

## Возможности v1.0

- открытый пул карт и его пополнение;
- отметка, кто забрал карту: отслеживаемый игрок или другой участник;
- группировка карт по игрокам;
- выбор одного или двух гонщиков от игрока для текущей гонки;
- номер текущей гонки и кнопка **«Следующая гонка»**;
- **«Новая партия»** с сохранением имён и настроек игроков;
- отдельный полный сброс приложения;
- справочник всех 36 гонщиков с оригиналом, переводом и уточнениями правил;
- миниатюры всех карт;
- сохранение состояния на устройстве;
- офлайн-работа после первого открытия.

## Принцип работы

Приложение помогает отслеживать пул, владельцев карт и состав текущей гонки, но не считает звёзды, места и победителей. Физическая игра остаётся источником истины.

## Развёртывание

Проект является статическим сайтом. Для Netlify:

- Build command: пусто;
- Publish directory: `.`;
- Production branch: `main`.

Перевод и справочные материалы неофициальные. Миниатюры подготовлены по материалам официального руководства для личного справочного использования.
