# Magical Athlete Companion

Мобильный русскоязычный помощник для настольной игры **Magical Athlete**.

**Приложение:** https://magical-athlete-guide.netlify.app

## Возможности

- настройка 2–6 игроков и пользовательские имена;
- режим одного или двух гонщиков от игрока;
- открытый пул и ручное добавление карт из справочника;
- статусы доступной, назначенной, участвующей, сыгранной и забранной другим игроком карты;
- группировка карт по игрокам;
- выбор участников текущей гонки с контролем лимита;
- завершение гонки с выбыванием сыгранных карт;
- отмена последнего завершения гонки;
- справочник всех 36 гонщиков с оригиналом, переводом и уточнениями правил;
- фильтры справочника по состоянию карты;
- качественные миниатюры из чистого источника руководства;
- сохранение партии на устройстве;
- офлайн-работа после первого полного открытия.

## Жизненный цикл карты

```text
Не добавлена → Доступна в пуле → У игрока → В гонке → Сыграна
                                   ↘ Другой игрок
```

После завершения гонки участвовавшие карты исчезают из открытого пула и списков игроков, но остаются в справочнике со статусом `Сыграна в гонке N`.

Приложение не считает звёзды, очки и победителей. Физическая игра остаётся источником истины.

## Развёртывание

Проект является статическим сайтом. Для Netlify:

- Build command: пусто;
- Publish directory: `.`;
- Production branch: `main`.

Перевод и справочные материалы неофициальные. Изображения используются как вспомогательные миниатюры для распознавания карт.
