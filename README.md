# Magical Athlete Companion

Мобильный русскоязычный помощник для настольной игры **Magical Athlete**.

**Приложение:** https://magical-athlete-guide.netlify.app

## Возможности v1.1

- открытый пул карт и его пополнение;
- явные состояния карты: вне партии, в пуле, у игрока, в гонке, сыграна, забрана другим игроком;
- распределение карт по игрокам;
- выбор одного или двух гонщиков от игрока для текущей гонки;
- завершение гонки с выводом участвовавших карт из пула и списков игроков;
- отмена последнего завершения гонки;
- фильтры справочника по состоянию карт;
- восстановление сыгранной карты в пул;
- номер текущей гонки;
- **«Новая партия»** с сохранением имён и настроек игроков;
- отдельный полный сброс приложения;
- справочник всех 36 гонщиков с оригиналом, переводом и уточнениями правил;
- миниатюры из чистого источника в повышенном разрешении;
- сохранение состояния и миграция данных предыдущей версии;
- офлайн-работа после первого открытия.

## Принцип работы

Приложение отслеживает пул, владельцев, состав текущей гонки и уже сыгранные карты. После завершения гонки её участники считаются использованными и больше не отображаются в пуле или у игроков. Приложение не считает звёзды, места и победителей: физическая игра остаётся источником истины.

## Развёртывание

Проект является статическим сайтом. Для Netlify:

- Build command: пусто;
- Publish directory: `.`;
- Production branch: `main`.

Перевод и справочные материалы неофициальные. Миниатюры подготовлены по материалам официального руководства для личного справочного использования.
