# Magical Athlete Companion v1.0.0

## Состав релиза

### Партия
- настройка 2–6 игроков;
- пользовательские имена игроков;
- режим одного или двух гонщиков от игрока;
- сохранение состояния на устройстве.

### Пул
- ручное добавление открытых карт из справочника;
- статусы «доступна», «забрал игрок», «забрал другой игрок»;
- пополнение пула без потери уже распределённых карт.

### Игроки и гонка
- группировка карт по владельцам;
- ограничение состава текущей гонки согласно выбранному режиму;
- отдельная вкладка только с активными гонщиками;
- неограниченный счётчик гонок;
- кнопка «Следующая гонка», очищающая только текущий состав.

### Управление данными
- «Новая партия» очищает пул и владельцев, но сохраняет настройки;
- полный сброс приложения требует двойного подтверждения;
- миграция состояния из предыдущей версии localStorage.

### Справочник
- 36 гонщиков;
- оригинальный текст и русский перевод;
- поиск по названиям, тексту и FAQ;
- уточнения спорных правил;
- миниатюры всех карт в общем оптимизированном sprite sheet.

### Offline/PWA
- установка на экран iPhone;
- кэширование интерфейса, данных и миниатюр;
- работа без интернета после первого успешного открытия.

## Намеренно не включено

- подсчёт звёзд и очков;
- победители и история мест;
- автоматический драфт и случайная раздача;
- моделирование trip, устранения и положения на трассе;
- автоматическое ограничение числа гонок.
